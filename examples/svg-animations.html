Getting Started with deck.js

How to Make a Deck

  1. Write Slides

    Slide content is simple HTML.

  2. Choose Themes

    One for slide styles and one for deck transitions.

  3. Include Extensions

    Add extra functionality to your deck, or leave it stripped down.

Animated SVG

Here:

     function(slide) {
           var a = $[deck]("svgAnimate", slide, ".svg1");
           $[deck]('addAnimation', slide, a.appear("#circleRed", 1200));
           $[deck]('addAnimation', slide, a.appear("#circleBlue", 300));
           $[deck]('addAnimation', slide, a.appear("#circleGreen", 400));
     }

And now, for more advanced stuffs

Animated SVG, multiple stuffs per step

Here:

     function(slide) {
           $[deck]('addAnimation', slide, {initAnimation: function() {$("pre", slide).removeClass("showAnyway")}, undoAnimation: function() {$("pre", slide).removeClass("showAnyway")}, doAnimation: function() {$("pre", slide).addClass("showAnyway")}});
           var a = $[deck]("svgAnimate", slide, ".svg1");
           $[deck]('addAnimation', slide, a.appear("#circleBlue, #circleGreen", 600));
           $[deck]('addAnimation', slide, a.disappear("#circleBlue, #circleGreen", 900));
           $[deck]('addAnimation', slide, [a.appear("#circleRed", 1200), a.appear("#circleBlue", 300)]); // for finer control (e.g., different durations)
           $[deck]('addAnimation', slide, [a.disappear("#circleRed", 300), a.disappear("#circleBlue", 900)]);
           $[deck]('addAnimation', slide, a.and(a.appear("#circleGreen", 400), a.appear("#circleRed", 400))); // useless to use "and" (see above for alternatives)
           $[deck]('addAnimation', slide, a.disappear("#circleRed, #circleGreen", 100));
           $[deck]('addAnimation', slide, [a.appear("#circleBlue", 1000), a.withDelay(1000, a.appear("#circleRed, #circleGreen", 300))]);
     }
